# Quick Start

This page has moved to the main documentation:

**[Getting Started →](https://nisugi.github.io/vellum-fe/getting-started/)**

The 30-second version:

```bash
# Via Lich (start Lich first)
vellum-fe --port 8000 --character YourName

# Direct eAccess (no Lich; password prompted securely)
vellum-fe --direct --account YOUR_ACCOUNT --character YourName --game prime
```

Config lives in `~/.vellum-fe/` on all platforms. Type `.help` in-game for
client commands, `.menu` for the menu, `.settings` for the settings editor.
